# Generic UI Components

This directory contains reusable, non-product-specific UI primitives for The Crafted Nest demo.

## Components

- Button: primary, secondary, and ghost actions
- Input: text entry field with optional error state
- Textarea: multi-line text entry
- Badge: status and label chips
- Container: page-width wrapper with responsive padding
- Section: vertical spacing wrapper for content sections
- Heading: semantic heading with tone and size variants
- Card: simple elevated content container
- Dialog: lightweight overlay container for modals or panels
- Breadcrumb: simple navigational trail
- Pagination: generic page navigation control

## Design rule

These components should remain generic and composable. Product-specific UI should be built in feature folders, not in this shared layer.
