# Layout Components

This directory contains the shared shell of the storefront experience.

## Components

- Navbar: responsive top navigation with desktop and mobile variants
- MobileNavigation: accessible slide-in style navigation for smaller screens
- Footer: structured site footer with premium spacing and typography
- Container: shared page-width wrapper used by the layout system
- PageLayout: composition layer for consistent page structure

## Design intent

These components are intentionally generic, responsive, and accessible. They provide the foundation for future pages without introducing any business logic.
