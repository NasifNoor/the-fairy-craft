# Willow & Co. Furnishings (Demo)

This scaffold follows a feature-based architecture for a Next.js App Router project.

## Folder purposes

- app/: Next.js App Router entry points, route groups, layouts, and global styles.
- components/ui/: Reusable presentational primitives such as buttons, cards, dialogs, and inputs.
- components/layout/: Shared layout pieces such as headers, footers, and page shells.
- components/sections/: Reusable page sections such as hero, featured collections, testimonials, and CTA blocks.
- features/home/: Home page experience and its local UI modules.
- features/catalog/: Product listing, filtering, and category browsing experience.
- features/product/: Product detail experience and related UI modules.
- features/contact/: Contact and inquiry experience.
- features/testimonials/: Testimonial and social proof experience.
- features/faq/: Frequently asked questions experience.
- features/cart/: Cart experience and related UI modules.
- features/checkout/: Checkout flow experience.
- lib/: Shared utilities, formatting helpers, motion helpers, and SEO helpers.
- data/dummy/: Dummy content and placeholder data for the demo experience.
- content/: Brand copy, marketing content, and SEO content files.
- hooks/: Reusable React hooks.
- types/: Shared TypeScript types and interfaces.
- services/: Data access and integration boundaries for future APIs or databases.
- config/: Project configuration and app-wide settings.
- styles/: Shared styling primitives or theme extensions.
- public/: Static assets such as images, icons, and fonts.

## Notes

- No business logic is implemented in this scaffold.
- Files are intentionally empty or documentation-only placeholders.
